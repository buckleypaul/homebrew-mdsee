# Homebrew Tap for mdsee

A Homebrew tap for [mdsee](https://github.com/buckleypaul/mdsee) - a simple markdown viewer for macOS with live reload.

## Installation

```bash
brew tap buckleypaul/mdsee
brew install mdsee
```

Or install directly:

```bash
brew install buckleypaul/mdsee/mdsee
```

## Usage

```bash
mdsee path/to/file.md
```
